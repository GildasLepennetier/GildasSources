INSTALLATION

<p>GildasSources does not require an installation. Just add the directory in the PATH, and give the execution rights.</p>
<ul>
<li>Exemple of commandes:</li>
<li>echo -e "#GildasSources\nPATH=$PATH:$(pwd)" >> ~/.bashrc</li>
<li>sudo chmod +x *</li>
</ul>
